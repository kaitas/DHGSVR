About

自己紹介

AICU Inc. / AICU Japan 株式会社所属のフルスタック／AI エンジニア。 Web プロダクトの初期立ち上げからグロースまで、情報設計・実装・計測を一貫して担当します。 ドキュメント化とデザインシステム整備で、チームが安心して改修できる土台づくりを重視しています。

  • UI アーキテクチャとデザインシステム設計
  • TypeScript / React / Next.js / Node.js / Python
  • LLM エージェント・RAG 設計と評価、GAS 自動化
  • アクセシビリティ・パフォーマンス改善、計測と改善サイクル
