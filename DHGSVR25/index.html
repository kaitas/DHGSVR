About

この講義について

デジタルハリウッド大学大学院「先端テクノロジー特論：人工現実」では、GitHub Pages を活用した Web ポートフォリオの制作、 AI ツールの活用、VR/AR などの人工現実技術について学んでいます。

  • GitHub / GitHub Pages の基本操作
  • HTML / CSS による Web 制作
  • AI ツールを活用したコンテンツ制作
  • VR / AR / XR 技術の理解と応用
